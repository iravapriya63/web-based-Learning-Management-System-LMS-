# web-based-Learning-Management-System-LMS-
Project: Learning Management System (LMS)

Tech Stack: Java 17, Spring Boot, MySQL/PostgreSQL, Spring Security, JUnit, Apache POI

Description:
Developed a web-based Learning Management System (LMS) at Georgia State University to manage online courses, assessments, and student performance tracking. Implemented a role-based system supporting Admin, Instructor, and Student users, enabling secure login, profile management, and differentiated access control.

Key Features:

User Management: Secure registration/login, role-based access control, and user profile updates.

Course Management: Instructors can create courses with media content, manage enrollments, and track attendance using OTP verification.

Assessments & Grading: Creation of quizzes and assignments; automated submission, grading, and feedback for students.

Performance Tracking: Dashboard to monitor student progress via quiz scores, assignment completion, and attendance.

Notifications: System and email notifications for enrollments, grades, and course updates.

Reporting & Analytics: Generate Excel-based performance reports and visualize student progress with charts.

Technical Highlights:

Backend REST APIs built using Java Spring Boot

Database management using MySQL/PostgreSQL

Spring Security for authentication and role-based access

Unit testing using JUnit

Apache POI for Excel report generation

Project managed and built with Maven
